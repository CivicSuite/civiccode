## Welcome To CivicCode Planning

CivicCode is the planned municipal code and ordinance-access module for
CivicSuite. The current repository is scaffold only. The first planning work is
Milestone 0: reconcile the upstream spec, queue ADRs, and define the test-first
runtime path.

Question for contributors: what municipal-code source formats should the first
importer support?

## Legal Advice Boundary

CivicCode must answer code questions with citations, but it must not provide
legal advice. We need clear resident-facing language for ambiguous questions,
stale sources, and staff interpretation routing.

Question for clerks and attorneys: what wording would your city be comfortable
showing to residents?

## CivicClerk Handoff

CivicClerk can record ordinance and resolution adoption events. CivicCode needs
a receiving contract before it can mark code stale or ingest adopted language.

Question for implementers: what fields must be in the first adoption-event
payload?
