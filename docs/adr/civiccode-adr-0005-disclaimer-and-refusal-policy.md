# CivicCode ADR-0005: Disclaimer And Refusal Policy

## Status

Proposed.

## Context

CivicCode can explain municipal code but must not provide legal advice. Resident
answers need consistent disclaimer and refusal language.

## Decision

Status: Open Question - pending human decision.

## Consequences

Prompt, API, and UI tests must prove that legal-advice boundaries are visible
and that ambiguous questions route to staff.
