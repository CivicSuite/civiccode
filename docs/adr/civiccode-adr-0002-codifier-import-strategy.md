# CivicCode ADR-0002: Codifier Import Strategy

## Status

Proposed.

## Context

Cities publish code through Municode, American Legal, Code Publishing, General
Code, local PDFs, web pages, and exports. The MVP must choose an import path.

## Decision

Status: Open Question - pending human decision.

## Consequences

The first runtime import milestone must follow the accepted source strategy and
must make unsupported sources actionable rather than silently ignored.
