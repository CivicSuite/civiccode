CivicCode
=========

Municipal code and ordinance access for the CivicSuite product family.

Current status
--------------

As of 2026-04-27, this repository is a scaffold only. No runtime application
code has shipped. There is no FastAPI app, no database schema, no frontend,
and no LLM workflow in this repo yet.

CivicCode is the next planning lane because CivicZone needs an authoritative
municipal-code source before zoning runtime work begins.
